# Getting Started

## Login & Signup
- Navigate to the Intellosync CLM login page.
- Sign up using your email or Single Sign-On (SSO) methods.
- Set up multi-factor authentication (MFA) for added security.
- If you forget your password, use the ‘Forgot Password’ option to reset it.

## Dashboard Overview
- The dashboard provides a **real-time summary** of your contracts and activities.
- Key sections include:
  - **Active Contracts** – View ongoing agreements.
  - **Pending Approvals** – Contracts requiring action.
  - **Recent Activities** – Track document modifications.
